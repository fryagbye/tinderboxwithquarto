Mermaid flowchart

This file can generate Mermaid flowchart from notes, links.

  • Container notes -> Subgraphs ( pMFSubgraph )
  • Non Container notes -> Nodes ( pMFNode)

Work Flow is a sample of Mermaid flowchart.
Please check it in Map view.

You need to use pre-release version chromeheadlessmodeissue

How to use

  1. You make a container noto for Flowchart and set pMFContainer.

    1. You make notes ( pMFNode) and links.

      1. You choose a direction for drawing like TD or LR.($M_Direction)

        1. Run the stamp ( Mermaid: Make a Flowchart ).

          1. Prototypes of container notes has a child note change from pMFNode to pMFContainer automatically by the stamp.

            1. The note (pMFContainer) has the code for mermaid flowchart.


Supported shapes

  • rectangle (<- normal and other shapes)
  • rounded
  • oval
  • lozenge
  • diamond
  • hex

Supported line types (Priority)

  • normal –> (4th)
  • bold ==> (3rd)
  • dotted -.-> (2nd)
  • invisible ~~~ (1st)

Supported arrow types

  • arrow only

Example

```{mermaid}

%%| label: fig-flowchart

%%| fig-width: 5

flowchart TB 

<p>A["Christmas"] -->|"Get money"| B["Go shopping"]</p>
<p>B["Go shopping"] -->  Sub1["Let me think"]</p>
<p>C["Laptop"]</p>
<p>D["iPhone"]</p>
<p>E["Car"]</p>

subgraph Sub1[“Let me think”] 

<p>C</p>
<p>D</p>
<p>E</p>

end

subgraph Sub2[“Subgraph”] 

<p>A</p>
<p>B</p>
<p>Sub1</p>

end 

<p>Sub1["Let me think"] -->|"One"| C["Laptop"]</p>
<p>Sub1["Let me think"] -->|"Two"| D["iPhone"]</p>
<p>Sub1["Let me think"] -->|"Three"| E["Car"]</p>
```
